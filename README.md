# Order Table With Telegram Sync

Google Apps Script для таблиці електроцеху.

Що робить:
- збирає вибрані замовлення з листа `Замовлення`
- відправляє пакет менеджеру в Telegram
- відправляє інженеру підтвердження пакета і зміни статусів
- формує лист `Документ для підпису`
- намагається відправити PDF менеджеру й інженеру
- якщо прямий `sendDocument` падає, відправляє посилання на PDF з Google Drive

## Структура таблиці

Лист `Замовлення`:
- `A` ID
- `B` Дата
- `C` Назва
- `D` Фірма виробник
- `E` Каталожний номер
- `F` Фото
- `G` Кількість
- `H` Терміновість
- `I` Зауваження
- `J` Надіслати
- `K` Стан замовлення
- `L` Пакет

Статуси:
- `Отримано`
- `В роботі`
- `Видано`

## Налаштування

1. Відкрийте Google Sheets і Apps Script.
2. Завантажте `Code.gs` та `appsscript.json`.
3. У `Script Properties` задайте:
   - `TELEGRAM_BOT_TOKEN`
   - `TELEGRAM_MANAGER_CHAT_ID`
   - `TELEGRAM_ENGINEER_CHAT_ID`
4. Створіть installable trigger:
   - function: `onEdit`
   - event source: `From spreadsheet`
   - event type: `On edit`

## Робочий сценарій

1. Заповніть рядки на листі `Замовлення`.
2. Поставте чекбокси в колонці `J`.
3. Запустіть меню `Електроцех` → `Надіслати вибрані замовлення`.
4. Скрипт:
   - групує вибрані рядки за датою
   - присвоює пакетний ID
   - надсилає пакет менеджеру
   - надсилає інженеру повідомлення
   - записує пакет у колонку `L`
   - перебудовує лист `Документ для підпису`
   - відправляє PDF або fallback-посилання
